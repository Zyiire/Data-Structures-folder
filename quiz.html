Part 2 — Choosing the Right Data Structure
Q4
Data structure identification
For each scenario, identify the best data structure (Stack, Queue, Linked List, Priority Queue) and justify in one sentence.
a) A hospital ER — critical patients treated first regardless of arrival time.
b) A web browser's Back button history.
c) A printer spooler — jobs processed in the order received.
d) A playlist where songs can be inserted or removed from any position.
a) Priority Queue — dequeues by severity, not arrival order.
b) Stack — LIFO: most recently visited page is returned first when pressing Back.
c) Queue — FIFO: jobs are printed in submission order.
d) Linked List — supports O(1) insertion/removal at arbitrary positions without shifting.

Enumerations in Java
Write a Java class that: defines an enum called Season with SPRING, SUMMER, FALL, WINTER; stores all values in an array; iterates printing each value's name and ordinal index.
public class SeasonDemo {
  enum Season { SPRING, SUMMER, FALL, WINTER }

  public static void main(String[] args) {
    Season[] seasons = Season.values();
    for (Season s : seasons) {
      System.out.println(s.ordinal() + ": " + s.name());
    }
  }
}

Q11
Coding: Stack and Queue methods
Write a complete Java program that: (1) creates a Stack using ArrayDeque and pushes integers 1–5, then pops and prints each; (2) creates a Queue using LinkedList and enqueues "A", "B", "C", then dequeues and prints each.
import java.util.*;

public class StackQueueDemo {
  public static void main(String[] args) {
    // Stack (LIFO)
    Deque<Integer> stack = new ArrayDeque<>();
    for (int i = 1; i <= 5; i++) stack.push(i);
    while (!stack.isEmpty())
      System.out.println(stack.pop()); // 5, 4, 3, 2, 1

    // Queue (FIFO)
    Queue<String> queue = new LinkedList<>();
    queue.offer("A"); queue.offer("B"); queue.offer("C");
    while (!queue.isEmpty())
      System.out.println(queue.poll()); // A, B, C
  }
}
